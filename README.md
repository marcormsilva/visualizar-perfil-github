# Visualizar Perfil Github

Este projeto permite buscar e visualizar perfis de usuários do Github, exibindo informações como avatar, biografia, seguidores, seguindo e os 10 últimos repositórios públicos.

## Funcionalidades
- Busca de usuário Github por nome
- Exibição de avatar, nome, bio, seguidores e seguindo
- Listagem dos 10 últimos repositórios, mostrando:
  - Nome
  - Estrelas
  - Forks
  - Watchers
  - Linguagem principal
- Layout responsivo e moderno
- Busca pode ser feita ao clicar no botão ou pressionar Enter

## Tecnologias Utilizadas
- HTML
- CSS (com variáveis e grid)
- JavaScript (ES6)

## Como usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```
2. Abra o arquivo `index.html` em seu navegador.
3. Digite o nome de usuário do Github e pressione Enter ou clique em "Buscar".
